Add 10 components that were documented on the docs site but missing from the CLI registry: carousel, combobox, command, dropdown-menu, hover-card, pagination, resizable, select, table, and tree. `npx @sanring/cli add <name>` now works for all of them. Also adds two new shared helpers (`collection-controller`, `collection-state`) used by combobox and command.
